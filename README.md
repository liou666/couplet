## 功能

- [x] 😊 桌面直接展示春联
- [x] 🎨 自定义字体和样式
- [x] 🚀 跨平台（Windows、macOS、Linux）
- [x] 🌙 支持深色模式
- [ ] 🌍 多语言支持

## 开发

```bash
# 1. 克隆该仓库;
git clone https://github.com/liou666/couplet.git

# 2. 安装依赖;
cd couplet
pnpm install

# 如果 Electron 安装失败，可以尝试使用淘宝镜像源👇:
# export ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/ && pnpm i

# 3. 启动服务
pnpm dev
```
